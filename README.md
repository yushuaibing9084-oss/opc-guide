## 📋 指南内容

本指南涵盖从 0 到 1 开店的完整闭环：

| 阶段 | 内容 | 预计时长 |
|------|------|---------|
| 阶段一 | 公司合规准备（营业执照/银行账户/域名） | 1-3 天 |
| 阶段二 | **ICP备案**（最关键，提前启动！） | 10-20 天 |
| 阶段三 | 注册开通微信小店 | 1-3 天 |
| 阶段四 | 申请教育/知识付费类目资质 | 1-5 天 |
| 阶段五 | 商品上架、自动发货配置 | 1-3 天 |

## ⚠️ 最重要的 3 件事

1. **营业执照经营范围**：写「教育咨询」，不要写「教育培训」，否则备案会被要求办学许可证
2. **ICP 备案第一天就启动**，不要等开店后再办，会白白多等 3 周
3. **商品类型选「虚拟商品」**，设置自动发货，否则买家付款后收不到课程
